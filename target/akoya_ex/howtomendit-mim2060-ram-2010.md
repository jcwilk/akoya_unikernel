# Medion Akoya EX (MIM2060) — memory upgrade note (HowToMendIt)

**Source URL:** https://howtomendit.com/answers?id=304145  
**Retrieved:** 2026-06-26  
**Akoya EX confirmation:** Accepted answer explicitly names “The Medion Akoya EX (MIM2060)”.

## Context

- User thread about installing a 1 GB PC2700 DDR333 SO-DIMM in a **Medion MIM2060** notebook (January 2010).
- Accepted answer (Erik, March 2010) ties MIM2060 to the Akoya EX product name.

## Facts stated for Medion Akoya EX (MIM2060)

| Field | Value |
|-------|-------|
| Internal designation | MIM2060 |
| Maximum total RAM | 1 GB (2 × 512 MB) |
| Memory type | PC2700 DDR |

## Notes

- Answer explains that a single 1 GB module is **not supported** on this machine (blank screen / boot failure), consistent with a 512 MB-per-slot maximum.
- Corroborates Offtek maximum-capacity data under the **MIM2060** internal name not present on the Offtek page.
